Get Request
Model.find -> Fetch All
Model.findOne -> Fetch a single item based on the parameter
Model.findById -> Fetch a single item based on the id

Update Request
Model.findOneAndUpdate -> Fetch and update a single item based on the parameter
Model.findByIdAndUpdate -> Fetch and update a single item based on the id

Delete Request
Model.findByIdAndDelete -> Fetch and delete a single item based on the id
Model.findByIdAndRemove -> Fetch and remove a single item based on the id
Model.findOneAndDelete -> Fetch and delete a single item based on the parameter
Model.findOneAndRemove -> Fetch and remove a single item based on the parameter
